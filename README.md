# NoteApp

A note keeping app.

---

## Setting up

> npm i

> npm start

Check out `log` dir for detailed logs and console for quick logs

Check [Wiki](https://github.com/vedant11/NoteApp/wiki) for extras

---

### <u>Tech stack</u> :

> react.js, express.js, node-pg (connector), postgresDB
